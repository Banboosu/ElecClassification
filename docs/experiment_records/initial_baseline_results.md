## 1. 数据集概况

原始数据文件为 `data/raw/最新多.csv`，字段包括充电站记录标识、类别标签和充电功率序列。

| 项目 | 数值 |
|---|---:|
| 原始记录数 | 46,535 |
| 唯一记录标识数 | 46,535 |
| 标签 0 | 12,115 |
| 标签 1 | 10,020 |
| 标签 2 | 12,965 |
| 标签 5 | 11,435 |
| 长度小于 18 的记录 | 10,614 |
| 最短/最长序列长度 | 1 / 816 |
| 平均序列长度 | 248.72 |

标签 5 被当前研究协议定义为无效或不完整数据，不作为故障分类类别。过滤标签 5 和长度小于
18 的序列后，共保留 35,099 条三分类记录，其中标签 0、1、2 分别为 12,115、10,020、
12,964 条。有效数据中未发现完全重复的功率序列。

标签 5 的平均序列长度仅为 5.90，而标签 0、1、2 的平均长度分别为 264.32、375.55、
350.29。旧四分类实验可能利用序列长度这一简单特征识别标签 5，因此不能直接与过滤后的三分类
实验比较。

## 4. 结果解释限制

旧 TCN 与初始 MOMENT 在类别集合、预处理和评估协议上均不一致，所以 97.54% 与 37.35%
不能用于得出 TCN 优于 MOMENT 的结论。初始结果只用于说明问题和确定后续优化方向。

从本记录之后，正式对比实验统一采用：

1. 相同的标签过滤、最小长度、最大长度与归一化方式；
2. 固定随机种子的分层 train/validation/test 划分；
3. Accuracy、Balanced Accuracy、Macro/Weighted Precision、Recall 和 F1；
4. 验证集用于模型选择，测试集只用于最终评估；
5. `zero_division=0`，避免无预测类别造成指标虚高。

## 5. 论文引用建议

该组结果可放在“初步实验与问题分析”章节，用于说明直接迁移预训练模型并不能自动获得良好
性能，以及统一实验协议的必要性。正式模型对比表应使用后续统一协议重新运行所得的结果。
